# BOP Matrix | X0X Drum & Bass

## Overview
A hybrid groovebox combining a 4-track drum machine with a monosynth bassline. Designed for sketching Drum & Bass, Jungle, and Breakbeat ideas, it features a unified interface for rhythm and bass.

## Unique Features
- **Hybrid Workflow:** Simultaneous control over drums and a synth voice.
- **Rhythm Core:** 16-step sequencer for Kick, Snare, Closed Hat, and Open Hat.
- **Bass Synth:** Independent 303-style bass voice with its own filter and drive controls.
- **Mixer & FX:** Dedicated volume faders for each track and global Delay/Reverb sends.
- **Glissando Keyboard:** Integrated keyboard for manual bass performance.

## Architecture
- **Drums:** Sample-based/Synth-based drum voices (Tone.MembraneSynth, NoiseSynth, MetalSynth).
- **Bass:** MonoSynth with Filter Envelope and Distortion.
- **FX:** Global Reverb and Feedback Delay.

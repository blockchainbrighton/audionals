# Suggested Enhancements for Patchbox

## Functionality
- **More Mod Slots:** Expand the matrix to include more destinations (Reverb Mix, Pan, Osc Mix).
- **Logic Modules:** Add mathematical modules (Add, Multiply) to combine LFOs for complex shapes.

## Creative Utility
- **Visual Cables:** Optionally render "virtual cables" on top of the matrix for a more modular feel.
- **Random Patch:** A button to randomly populate the matrix connections for happy accidents.

## User Experience
- **Matrix Zoom:** Ability to "solo" or zoom into specific modulation routings for fine-tuning.
- **Preset Sharing:** Generate a URL parameter string to share specific patches.

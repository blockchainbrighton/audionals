# Synthwave Synth (Legacy)

## Overview
The prototype for the "Neon Horizon" synth. It sets up the polyphonic saw-wave architecture and the initial "Macro" control concept.

## Unique Features
- **Polyphonic Engine:** Basic polyphonic oscillator setup.
- **Macros:** Early implementation of the Brightness/Motion controls.

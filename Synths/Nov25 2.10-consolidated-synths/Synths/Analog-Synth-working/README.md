# Analog Synth (Legacy)

## Overview
The foundation for the "BOP Matrix" series. This is a functional subtractive synthesizer prototype featuring the core building blocks of analog synthesis: Oscillators, Filters, and Envelopes.

## Unique Features
- **Core Analog Path:** Oscillator -> Filter -> Amp.
- **Direct Controls:** straightforward sliders for ADSR and Filter cutoff.
- **Prototype Status:** Represents an earlier iteration of the engine found in the "Gemini3" folder.

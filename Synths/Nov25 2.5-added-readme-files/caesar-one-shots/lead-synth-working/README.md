# Lead Synth (Legacy)

## Overview
The working prototype for the Expressive Lead synth. It defines the mono/lead playing style and the basic effects chain structure.

## Unique Features
- **Lead Voicing:** Monophonic priority logic.
- **Basic FX:** Early implementation of the delay/reverb chain.

## Status
*Note: This is a legacy/working version. For the enhanced version with pitch bend, stereo widening, and updated visuals, see `Gemini3-oneshots/Lead-Synth-nice-keyboard-and-action`.*

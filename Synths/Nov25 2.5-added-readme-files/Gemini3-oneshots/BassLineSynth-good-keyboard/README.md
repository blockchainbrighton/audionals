# ORD-303 Bassline Matrix

## Overview
A faithful recreation of the legendary acid bass synthesizer. The ORD-303 focuses on the squelchy, resonant sound of the TB-303, complete with a 16-step sequencer that handles Note, Gate, Accent, and Slide programming.

## Unique Features
- **Acid Sequencer:** Classic step sequencer with per-step Accent (velocity/filter boost) and Slide (portamento).
- **Filter Character:** 24dB Lowpass filter tuned for "acid" resonance sweeps.
- **Distortion:** Built-in drive control to push the sound into aggressive territories.
- **Visualizer:** Real-time waveform visualization.
- **Presets:** Curated selection of bass and acid patches.

## Architecture
- **Oscillator:** Switchable Sawtooth / Square.
- **Filter:** Resonant Lowpass with Envelope Modulation and Decay.
- **Sequencer:** 16-step grid with intuitive LED-style feedback.

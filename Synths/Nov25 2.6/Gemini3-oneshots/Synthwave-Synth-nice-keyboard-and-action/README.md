# NEON HORIZON | Synthwave Poly

## Overview
A retro-futuristic synthesizer designed specifically for Synthwave, Outrun, and 80s pop. It features "fat" oscillators, lush chorus and reverb, and a simplified "Macro" control scheme for instant vibe.

## Unique Features
- **Macros:** Three main controls (Brightness, Motion, Space) that morph multiple parameters simultaneously for easy performance.
- **Retro Aesthetic:** CRT scanline overlay, neon color scheme, and a wireframe sun visualizer.
- **Fat Oscillators:** Uses "fatsawtooth" and "fatsquare" types for a thick, detuned analog sound.
- **Arpeggiator:** Integrated latching arpeggiator for driving basslines.

## Architecture
- **Engine:** PolySynth (10 voices).
- **Oscillators:** Multi-layered spread oscillators.
- **FX:** Chorus (Dimension D style), PingPong Delay, Hall Reverb.
- **Visuals:** Three.js powered Outrun-style landscape.

# BassLine Synth (Legacy)

## Overview
An early prototype of the Acid Bass synthesizer. It establishes the core sequencer logic and the sawtooth/square oscillator switching mechanism.

## Unique Features
- **Step Sequencer:** Basic implementation of the 16-step sequencer.
- **Filter Controls:** Fundamental Cutoff and Resonance sliders.

*Note: This is a legacy/working version. For the full featured version with slide/accent logic and improved UI, see `Gemini3-oneshots/BassLineSynth-good-keyboard`.*

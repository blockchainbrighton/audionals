♥ Mothy G ♥

A Love-Inspired JavaScript Synthesizer

About Mothy G

Mothy G is a fully functional, VST-ready synthesizer built entirely in JavaScript using Tone.js and Web Components. It can run standalone in a browser or be embedded as a component in other JavaScript applications.

Features

  • Dual oscillator synthesis with multiple waveforms
  • Resonant filter with envelope modulation
  • Independent amplitude and filter envelopes (ADSR)
  • Built-in effects: Reverb, Delay, Chorus
  • 15 expressive love-themed presets
  • Virtual keyboard with computer keyboard support
  • Preset save/load with localStorage persistence
  • Import/export preset functionality

Keyboard Controls

Use your computer keyboard to play notes: A W S E D F T G Y H U J K O L P ; ' ]

Or click/touch the virtual keyboard with your mouse or touchscreen.

Integration

To use Mothy G in your own project, simply import the component:

<script type="module" src="./src/MothyG.js"></script>
<mothy-g></mothy-g>

Technical & Rarity Specifications

Rarity Distribution

  • Ultra-Rare (≈1%): Top-tier animation & patch combinations that appear only once.
  • Rare (≈10%): Limited combinations with ≤10 total mints each.
  • Uncommon/Common (≈89%): Balanced for accessibility while ensuring all are distinct.

All rarity logic is deterministic and verifiable entirely on-chain.

Core Technology

  • 100% On-Chain: All assets are inscribed on Bitcoin. No IPFS or external servers required.
  • Opus-Powered: Uses the advanced Opus codec for hi-fi audio with minimal data footprint.
  • Open-Source Ethos: Assets use a CC-BY-SA-Web3 license, and the Opus File Generator is free.
